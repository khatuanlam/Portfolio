Lập trình Java

Là ngôn ngữ chủ đạo trong học tập ở trường.

Thành thục Java ở mức học tập trong trường với những kĩ năng chính:

  • Sử dung được phân luồng trong Java (môn Hệ điều hành)
  • Kĩ năng lập trình (môn Kĩ năng lập trình)
  • Các cấu trúc dữ liệu cơ bản, các giải thuật sắp xếp tìm kiếm (môn Cấu trúc dữ liệu và Giải thuật)
  • Sử dụng OOP ở mức tốt, hiểu được các loại thuộc tính trong Java. Sử dụng OOP xử lý các bài toán trong AI.

Lập trình ứng dụng mobile (Retrofit, Call API, OKHTTP, Firebase)

Là công cụ làm bài tập lớn trong môn Lập trình nâng cao.

  • Biết cách tạo ra một ứng dụng dựa theo mẫu thiết kế sẵn, các chức năng cơ bản như tạo danh mục, tạo thanh tìm kiếm đổ xuống, tạo box
  • Biết cách sử dụng API, và kéo dữ liệu về, Parse một trang web để lấy dữ liệu, truyền dữ liệu vào những box có sẵn
  • Hiểu được giao diện trên mobile
  • Thực hiện lưu trữ database trên kho Firebase của Google

HTML, CSS, JavaScript, ReactJS, NodeJS

Là công cụ làm BTL một số môn, trong quá trình thực tập.

  • Tạo ra được một trang web thông tin (HTML, CSS, JS) hiểu được các thuộc tính cũng như cách viết 1 trang web hoàn chỉnh dùng để nộp báo cáo BTL.
  • Sử dụng được Bootstrap để tạo ra trang web nhanh chóng.
  • Có kinh nghiệm sử dụng ReactJS (Front-end) và NodeJS (Back-end MongoDB) trong một dự án thực tế.
